# GharPE Backend

NestJS backend API for GharPE application.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## API Endpoints

- `GET /` - Hello World message
- `GET /health` - Health check endpoint

## Default Port

The application runs on port `3001` by default. You can change this by setting the `PORT` environment variable.
